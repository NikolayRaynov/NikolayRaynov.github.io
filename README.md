Welcome to my personal portfolio repository! This project serves as a centralized hub to showcase my technical skills, professional background, and the software development projects I've built as an aspiring **.NET Developer**.

## 🌐 Live Website
Check out the live version here: [https://nikolayraynov.github.io/](https://nikolayraynov.github.io/)

---

## 🛠 Tech Stack & Tools
This portfolio was built using modern web technologies to ensure a smooth and professional user experience:

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Frameworks:** Bootstrap 5 (for responsive layout)
* **Icons:** Font Awesome & Bootstrap Icons

---

## ✨ Key Features
* **Fully Responsive:** Optimized for all screen sizes, from mobile phones to ultra-wide monitors.
* **Interactive UI:** Smooth transitions and animations that enhance user engagement without sacrificing performance.
* **Dynamic Sections:**
    * **Skills:** Visualized with progress bars to show proficiency levels.
    * **Resume:** A structured timeline of education.
    * **Portfolio:** A filterable gallery of my latest coding projects.
* **Contact Integration:** Easy access to my professional social media and email.
